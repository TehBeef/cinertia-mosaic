Mosaic

A professional NDI® multiviewer for Windows and macOS. Every source on your network, arranged exactly how you want it.

⬇  Download for Windows ⬇  Download for macOS
Windows 10/11 · macOS 12+ (Apple Silicon & Intel) · Free

Built for show day

Automatic NDI® discovery

Every NDI® source on the network appears in the sidebar — click to add it to the wall. Duplicate one source into several tiles for wide shot + detail.

Free-form canvas

Drag, resize, and layer tiles anywhere, or snap to a grid. One-click broadcast layouts: 2×2, 3×3, 4×4, 1+side, 2+8.

GPU view controls per tile

Scroll to zoom, drag to pan, crop to a detail, rotate — instant, on the GPU, at full source frame rate. Direct3D on Windows, Metal on macOS.

Instant profiles

Save a complete look — sources, layout, every tile's view — and switch in one click, one hotkey, or one Stream Deck button. Shared sources never reconnect.

Multi-monitor canvases

Open extra output windows, each a full canvas: windowed, fullscreen on any display, or frameless. A saved profile restores the whole multi-display look.

Stream Deck control

Native Bitfocus Companion module with live profile dropdowns and active-profile button feedback — plus a simple TCP protocol for anything else.

Clean, minimal chrome

Dark theme, hover-reveal controls, windowless mode. The video is the interface.

Show-day details

Rename tiles ("CAM 1 — STAGE LEFT"), name overlays, audio meters, low-bandwidth mode for small tiles, keep-display-awake, session autosave.

Install in a minute

Windows
Windows 10/11, 64-bit
  1. Download Mosaic-Setup-<version>.exe from the latest release and run it.
  2. When Windows Firewall asks, click Allow — NDI® needs network access.
macOS
macOS 12+ · Apple Silicon & Intel
  1. Download Mosaic-<version>.dmg from the latest release, drag Mosaic to Applications.
  2. First launch: right-click → Open (needed once).
  3. Click Allow on the Local Network permission — that's NDI® discovery.
